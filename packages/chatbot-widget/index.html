Coreforge Chatbot Widget

This demo page shows the embeddable chatbot widget. Click the chat bubble in the bottom-right corner to try it out.

Embed Code

Add this to any website:

<script src="https://cdn.example.com/chatbot-widget.iife.js"></script>
<script>
  CoreforgeChatbot.init({
    apiBaseUrl: '/api/chatbot',
    companyName: 'Your Company',
    primaryColor: '#2563eb',
    welcomeMessage: 'Hi there! How can we help?'
  });
</script>

Configuration Options

{
  apiBaseUrl: '/api/chatbot',      // Backend API URL
  companyName: 'Your Company',     // Brand name
  primaryColor: '#2563eb',         // Brand color
  position: 'bottom-right',        // or 'bottom-left'
  welcomeMessage: 'Hi!',           // First bot message
  title: 'Chat with us',           // Panel header title
  bookingEnabled: false,           // Enable booking flow
}
